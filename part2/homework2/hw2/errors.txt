2) Вместо создания нового массива строк создается только одна строка

СТРОКА 36
НЕПРАВИЛЬНЫЙ ВАРИАНТ: string new_word_list = *(new string[2 * capacity]);  
ПРАВИЛЬНЫЙ ВАРИАНТ: string *new_word_list = new string[2 * capacity]; 


3) Нужно присвоить элемент одного массива элементу другого массива

СТРОКА 40
НЕПРАВИЛЬНЫЙ ВАРИАНТ: new_word_list = word_list[index];  
ПРАВИЛЬНЫЙ ВАРИАНТ: new_word_list[index] = word_list[index]; 


4) Вместо очистки памяти, отведенной под массив,

СТРОКА 42
НЕПРАВИЛЬНЫЙ ВАРИАНТ: delete word_list;
ПРАВИЛЬНЫЙ ВАРИАНТ: delete [] word_list;


5) Нужно, чтобы один массив ссылался на другой массив.
